**DailyEnglish** is a modern, AI-powered English learning mobile application built with React Native and driven by on-device Machine Learning via TensorFlow.js. 

Unlike traditional language apps that rely on rigid static rules or heavy external server API requests, DailyEnglish brings the power of intelligence directly onto the user's device. It processes user performance data in real-time to dynamically shape the learning journey, offering smart quiz scoring and adaptive learning workflows.

✅ **Interface:** Modern, minimalist, and dynamic visual elements (Goodbye classic folder icons!)
✅ **Ideal for:** Language Learners, Mobile Developers, AI/ML Engineers, and Portfolio Showcase

---

## 🧠 Overview & Methodology

The application focuses on an intelligent feedback loop that optimizes how language retention is measured and rewarded:

1. **🤖 On-Device Machine Learning:** Utilizing local **TensorFlow.js Regression Models** to analyze user quiz metrics, processing response times, accuracy, and difficulty levels instantly without sending sensitive data to external servers.
2. **📈 Smart Quiz Scoring:** Instead of a simple percentage score, the app calculates a weighted, comprehensive user performance score that accurately reflects the user's current command over the language topic.
3. **🔄 Adaptive Learning Workflows:** The app interprets the ML model outputs to dynamically adjust upcoming quiz difficulties, vocabulary frequencies, and review schedules based on identified cognitive fatigue or mastery patterns.

---

## ⚙️ Key Features

* **AI-Powered Analytics:** On-device regression model execution for smart, contextual performance evaluation.
* **Modern & Elegant UI/UX:** A clean dashboard breaking away from outdated, folder-heavy interfaces, replacing them with dynamic components, progress highlights, and smooth transitions.
* **Smart Performance Insights:** Actionable feedback based on complex machine learning analytics transformed into human-readable suggestions.
* **Visual Progress Tracking:** Sleek progress bars, visual stats for mastered vocabulary tracks, and active learning streaks.
* **Privacy-First Architecture:** Local machine learning model deployment guarantees that user analytics and data never leave the phone.

## 🚀 Installation & Usage

Follow these steps to set up the project locally on your development machine:

### 1. Clone the repository
```bash
git clone [https://github.com/OmerFarukKarakoy/DailyEnglish.git](https://github.com/OmerFarukKarakoy/DailyEnglish.git)
cd DailyEnglish
```
### 2. Install required dependencies
```npm install```
Or if you use yarn:
```yarn install```

### 3. Configure iOS (Optional - If running on a Mac for iOS testing)
```
cd ios
pod install
cd ..
```

### 4. Run the Program
Make sure you have an Android Emulator, iOS Simulator, or a physical device connected.
For Android:
```npx react-native run-android```
For iOS:
```npx react-native run-ios```
